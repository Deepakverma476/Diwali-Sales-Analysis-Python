# 🎇 Diwali Sales Analysis Using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Diwali Sales dataset using Python. The objective is to analyze customer purchasing behavior during the festive season and generate meaningful business insights.

---

## 🎯 Objectives

* Clean and preprocess the dataset.
* Analyze customer demographics.
* Identify top-performing states and occupations.
* Study purchasing patterns.
* Generate business insights through data visualization.

---

## 🛠️ Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Information

The dataset contains customer information including:

* Gender
* Age Group
* Marital Status
* State
* Occupation
* Product Category
* Orders
* Amount Spent

---

## 📊 Analysis Performed

* Data Cleaning
* Missing Value Handling
* Gender Analysis
* Age Group Analysis
* Marital Status Analysis
* State-wise Sales Analysis
* Occupation Analysis
* Product Category Analysis
* Top Selling Products Analysis

---

## 📈 Key Insights

* Female customers contribute more to total sales than male customers.
* Customers aged **26–35 years** generate the highest revenue.
* Married women are the most valuable customer segment.
* Uttar Pradesh, Maharashtra, and Karnataka are among the highest revenue-generating states.
* Customers working in IT, Healthcare, and Aviation sectors spend the most.
* Food, Clothing, and Electronics are among the most popular product categories.
